# 1661. Average Time of Process per Machine

```mysql
SELECT
    aStart.machine_id, 
    ROUND(AVG(aEnd.timestamp - aStart.timestamp),3) AS 'processing_time'
FROM
    Activity aStart
JOIN Activity aEnd ON
    aStart.machine_id = aEnd.machine_id AND 
    aStart.process_id = aEnd.process_id AND
    aEnd.activity_type = 'end'
WHERE 
    aStart.activity_type = 'start'
GROUP BY machine_id;
```
